Code's genereren

De code wordt gegenereerd door een algoritme. De code wordt gegenereerd door een random string van 6 karakters te genereren, deze wordt gelijk naar de mentoren gemaild. Hierna wordt deze string vervolgens ge-hashd met een argon 2 algoritme en opgeslagen in de database. Door deze methode is het niet mogelijk om de code te achterhalen.
